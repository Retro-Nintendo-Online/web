Netplay Setup Guide

Step 1: Download the Emulator

Download Project64 Netplay a project by Cranky Supertoon.

Download

Step 2: Gather Your Games

Dump the games for the games you want to play. The Mario Party Trilogy, Mario Kart 64, and Super Smash Bros are all supported with other games such as Mario Golf and Mario Tennis too.

These dumped games should be placed in a folder that the Emulator can find, and should have the file extension .Z64, .V64, or .N64

Step 3: Set Up the Emulator

Project 64 should appear inside the .ZIP file from the Github. You can launch it using Project64.exe

Go to Files > Choose ROM Directory and select the folder where you placed your dumped Games.

Then go to Options > Configure Controller Plugin and select N-Rage Direct Input. Then click Configure to set up the controller for your games. Configure it to suit your needs. Then click OK.

Netplay Usage Guide

Step 1: Launch Your Game

Launch Project64.exe.

From here, you can double-click your game to access the netplay window.

The game window is supposed to be black. Use the pop-up window to host a party or connect to your fellow partiers.

Step 2: Connect to the Other Players

If you plan to host your own lobby for others to join, there is 2 way to do this. One is to use a dedicated server ran by AQZ the person who made the Netplay Support a reality. Or you can use a forwarded port and follow the Direct Connection instruction below. If you plan to use AQZ's dedicated server find the Dedicated Server part.

Dedicated Server

On the bottom right corner of the NetPlay window you will see a list of servers. Click on the one with the smallest ping notified in ms

Then copy the /join string of text and give it to your fellow partiers.

By pasting in the command They can join in on the fun.

Direct Connection

If you have a forwarded port i highly recommend using that instead of the dedicated server.

Typing /host followed by the port you have forwarded

Then the others need to type: /connect <hosts public IP>:<hosts forwarded port>

Step 3: Start the Game

Once everyone is together, you may start the party using /start. In case of crashing or desync, please redo the previous step.

Other Notes

Every player must have the same save file for saving to work. Be sure to select Replace Save Files from the emulator's Netplay tab to replace the saves before every game you play.

If a Save File is not the same you are going to experience a Desync

Player order is decided by the order of the netplay room. In case of desync or crashing, make sure to join back in the same order or change ports in-game.

Make sure everyone playing has only the controller they're using plugged in.

It's possible for multiple people to play from the same computer as long as the controllers they're using are plugged in.

It's possible to host a room for 4 other people as long as the host disables their controller in PJ64.

Lag is an unfortunate but necessary part of netplay. Just use the /lag command to increase input buffer and make the game smoother.

The emulator's Netplay tab includes Update Emulator, which will add new features such as new cheat codes and save files, as well as other suprises.

Frequently Asked Questions

Question 1: I already have PJ64, do I need to download it again?

Answer 1: This guide provides a version of this emulator geared specifically for netplay, and it's incompatible with other versions.

Question 2: How can I reduce netplay lag?

Answer 2: you can try increasing the lag via /lag, Closing any other programs using internet (web browsers, skype), making sure no one on the same network is using the internet (family, roommates), and finally Ensure it's actually netplay lag by having each player test FPS locallyQuestion 3: The game's graphics are bad! Troubleshooting

Question 1: I was playing a game and the screen went black! What do i do?

Answer 1: Quit out of the game and hope this game saves progress

Question 2: I started emulation in a netplay lobby, but the game showed up as a black screen!

Answer 2: Make sure everyone is using the same plugins as everyone else.

Question 3: Someone desynced!

Answer 3: A desync most likely means that you didn't all have the same cheat codes or the same save file. Replace.bat will replace your save to a 100% save file that has the same stuff so it will not desync.

Question 4: Why is the Netplay Window not showing?

Answer 4: You did install Project64 Netplay linked at the start of this thread right?
