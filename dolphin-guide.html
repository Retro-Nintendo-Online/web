Netplay Setup Guide

Step 1: Download the Emulator

Download Dolphin

Download

Step 2: Gather Your Games

Dump the games for the games you want to play. The Mario Parties, Mario Kart Double Dash, and Super Smash Bros Melee are all supported with other games such as Mario Golf Toadstool Tool and Mario Superstar Baseball too.

These dumped games should be placed in a folder that the Emulator can find, and should have the file extension .ISO

Step 3: Set Up the Emulator

Dolphin should appear inside the downloaded file from the Github. You can launch it using Dolphin.exe

If you want dolphin to run in a Portable mode, create a portable.txt file inside the Dolphin Directory

When Launching, double click the empty box and select the folder where you placed your dumped Games.

Then go to Controllers. Under GameCube Controllers select your controller from the drop-down menu next to Port 1. Click Configure to the right of the drop-down menu and configure your controls to suit your needs.

Netplay Usage Guide

Step 1: Launch Your Game

Launch Dolphin.exe.

From here, got to Tools > Start NetPlay.

The game window is supposed to be black. Use the pop-up window to host a party or connect to your fellow partiers.

Step 2: Connect to the Other Players

If you plan to host your own lobby for others to join, there is 2 way to do this. One is to use a traversal code. Or you can use a forwarded port and follow the Direct Connection instruction below. If you plan to use traversal follow the Traversal Server part.

Traversal Server

If you plan to use traversal, pick the Traversal Server option followed by the Host tab and double-click the game you want to host. Click Copy next to the IP and give it to the players who will join you.

Direct Connection

If you plan to host your own lobby for others to join, pick the Direct Connection option followed by the Host tab and double-click the game you want to host. Click Copy next to the room ID and give it to the players who will join you.

Step 3: Start the Game

Once everyone is together, the host may start the party using Start. In case of crashing or desync, please redo the previous step.

Other Notes

Player order is decided by the order of the netplay room by default. The host can change controller ports in the netplay window.

It's possible for multiple people to play from the same computer as long as the controllers they're using are plugged in and the host allows them use of multiple ports.

It's possible to host a room for 4 other people as long as the host disables their controller in Dolphin..

It's possible to host a room for 4 other people as long as the host disables their controller in PJ64.

Lag is an unfortunate but necessary part of netplay. The host may use theBuffer selection to increase input buffer and make the game smoother.

Possible desyncs aren't always truly desyncs, keep playing to make sure if there was truly a desync before restarting.

Direct connect may be used even when the traversal server is down, and for players who have trouble with the traversal server at all times it may be the only way to play. Forward a port if the traversal server isn't working for you.

Frequently Asked Questions

Question 1: I already have Dolphin, do I need to download it again?

Answer 1: You shouldnt have to if you run the auto updater.

Question 2: How can I reduce netplay lag?

Answer 2: you can try increasing the buffer, Closing any other programs using internet (web browsers, skype), making sure no one on the same network is using the internet (family, roommates), and finally Ensure it's actually netplay lag by having each player test FPS locally

Question 3: The game's graphics are bad! Can I make them look better?

Answer 2: The resolution and anti-aliasing of a game can be changed without issue, though it is not recommended as the emulation will require more resources.

Question 4: How do I use the GameCube Mic accessory over netplay

Answer 4: You can't.

Troubleshooting

Question 1: Whenever I try to join someone's traversal lobby, Dolphin just stalls and I can't connect!

Answer 1: Depending on how your network is configured, you may only be able to connect to lobbies hosted over Direct Connect. Either have the player unable to connect host the traversal lobby themselves or have the host forward a port and host over Direct Connect. If you need help forwarding a port, try following this guide.

Question 2: I was playing a game and the screen went black! What do i do?

Answer 2: Quit out of the game and hope this game saves progress

Question 3: I started emulation in a netplay lobby, but the game showed up as a black screen!

Answer 3: The black screen is caused by players not starting emulation at the same time. Ensure that no one in the lobby is running the game before the host starts netplay.

Question 4: Someone desynced!

Answer 4: A desync most likely means that the checksum on your games is not all the same since it should sync cheats and saves.

Question 5: How do I solve the error The program can't start because MSVCP140.dll is missing from your computer?

Answer 5: Download https://www.microsoft.com/en-us/download/details.aspx?id=48145 and run the file.
