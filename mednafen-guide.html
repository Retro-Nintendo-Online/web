Netplay Setup Guide

Step 1: Download the Emulator

Download Mednafen - RNO Edition

Download

Step 2: Gather Your Games

Dump the games for the games you want to play. Super Mario Bros, Bomberman, and Super Mario Kart are all supported with other games too.

These dumped games should be placed in a folder that the Emulator can find, and should have the file extension .NES, .SMC, or .SFC

Step 3: Set Up the Emulator

Mednafen should appear inside the downloaded file from the Github. You can launch it using medanfen-netplay.exe

After Launching, click File -> Open ROM followed by clicking Add. Click the button to select the path for your games. After you finish that click the big button that says Done.

Medanfen is a pain to set up Controllers with so for now Google it, but we will make it easier in the near future.

Netplay Usage Guide

Step 1: Launch Your Game

Launch medanfen-netplay.exe.

From here, go to File -> Open ROM Then Select your ROM Directory and click the game you choose to play.

Step 2: Connecting to Madenfen's Netplay Servers

Assumbing you want to play netplay which is why your probally here, on the right side of the GUI check the Netplay box. Then give yourself a Name. After that type in a Game Code. Every Medanfen Netplay game needs a seperate code so be sure to make it unique.

Step 3: Start the Game

After this People can start their game. Desyncing at the start is not a issue since medanfen syncs Save States and not Save Files

Other Notes

Player order is decided by the order of people clicking Start Medanfen room by default.

Hitting the letter T on your keyboard can give you access to medanfen's netplay console to type to others, and add commands.

Lag is an unfortunate but necessary part of netplay. You may use the/ping command to see how much ping the game is currently using.

Direct connect can't currently be used but we hope to adopt it soon once Cranky cracks Mednafen Server.

Frequently Asked Questions

Question 1: I already have Mednafen, do I need to download it again?

Answer 1: We use a custom version geared towards Netplay.

Question 2: How can I reduce netplay lag?

Answer 2: Closing any other programs using internet (web browsers, skype), making sure no one on the same network is using the internet (family, roommates), and finally Ensure it's actually netplay lag by having each player test FPS locally
